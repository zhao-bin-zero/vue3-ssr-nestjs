# vue3-ts-vite--nest-ssr

- Lightning-fast cold server start
- Instant hot module replacement (HMR) and dev SSR
- True on-demand compilation
- Use `vue3 vite2.0 typescript eslint SSR vuex vue-router element-plus scss`
- Use `node nestjs`

# Getting Started
- Nest dev
```bash 
npm i
npm run dev
```


- Express dev
```bash 
npm i
npm run dev2
```


- preview
```bash 
npm i
npm run preview
```


- build
```bash 
npm i
npm run build
```
